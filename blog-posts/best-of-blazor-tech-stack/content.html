Some time back, a guy named Vetri Maran kickstarted a blog named "Best of Blazor." Given that Blazor was still relatively new, it immediately caught my attention. However, after a while, the blog disappeared without a trace, leaving me in shock.

Fast forward to October 2023, and I have had the opportunity to resurrect this beloved blog. This time around, I aspire to retain the essence of the original blog while injecting fresh content, making it larger and better than before.

A heartfelt thank you goes to Blazor School and, specifically, Huy Tran for lending a helping hand in rebuilding this blog. Huy Tran is more than just a mentor; he's a knowledgeable guide and a true leader.

But let's return to the main focus of this inaugural post—the technology stack that brought "Best of Blazor" back to life:

  1. Blazor Server:
    Blazor forms the core of this blog's frontend, enabling the creation of interactive and dynamic web applications using C# instead of traditional frontend languages like JavaScript. It's fast, it's easy.

  2. GitHub Authentication:
    Utilizing GitHub for server-to-server authentication, we've ensured website security with minimal effort.

  3. Visual Studio:
    My primary code editor of choice, Visual Studio, was selected for its flexibility, extensive extension library, and seamless integration with .NET and Blazor development.

  4. Hostinger:
    For hosting, Hostinger proved to be a cost-effective choice at just $6 per month. It's stable, affordable, and boasts excellent support. If you're considering hosting, I highly recommend using our Best of Blazor referral link for a 20% discount here.

  5. highlight.js:
    I've employed highlight.js to bring color to the code. While it's not perfect, it's a fast and straightforward choice.

  6. SQLite:
    Choosing SQLite for data storage was a strategic move due to its lightweight nature. Unlike SQL Server, it doesn't demand extensive RAM (thus reducing hosting costs), yet it remains fast and easy to use. Plus, with Entity Framework, database management becomes a breeze.

With this technology stack, my aim was to create a blog that not only reflects my passion for Blazor but also provides you, the reader, with a seamless browsing experience.
